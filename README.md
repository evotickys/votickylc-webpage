# Página Web de Voticky Legal & Consulting # 

Este repositorio contiene el código fuente de la página web estática de Voticky Legal & Consulting.

# Descripción #

Página web de Voticky Legal & Consulting, firma boutique dedicada a la asesoría legal-regulatoria y de negocios, enfocada en empresas Fintech y organizaciones innovadoras.

# Stack Tecnológico #

HTML5

Tailwind CSS (vía CDN)

JavaScript (mínimo, para interacciones de UI)

# Despliegue #

Este sitio está diseñado para ser desplegado automáticamente en plataformas de hosting estático como Vercel, Netlify o GitHub Pages.
